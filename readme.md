# [3D Wine Cellar](https://cellar3d.vercel.app)

**Start Date:** June 2025

---

### Initial Refactor
* **Parametric Geometry**
  Racks are now fully configurable by rows, columns, depth. Spacing, lighting, and shelving wire-density adjust dynamically to fit dimensions.

---

### Subsequent Changes
* **Multi-Cellar Architecture**
  Added support for instantiating multiple independent cellar units within the global coordinate system.
* **Professional Aesthetic**
  Stripped legacy "game-like" textures in favor of a minimalist appearance.
* **State Persistence**
  LocalStorage to persist cellar configurations across sessions.
* **Smart Inheritance**
  New cellar instances copy the configuration of the preceding unit for seamless expansion.
